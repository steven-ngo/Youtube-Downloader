## Dependencies
`streamlit` `pytube`

## Run the App Locally
### Create conda environment
```
conda create -n app python=3.10.0
```
```
conda activate app
```
### Install Dependencies

Download requirements.txt file

Pip install libraries
```
pip install -r requirements.txt
```

###  Launch the app

```
streamlit run NationalDebt.py
```
